# Logistic Regression with CUDA

This project implements logistic regression training using CUDA C++ for GPU acceleration.

## Structure

logistic-regression-cuda/
├── src/
│ ├── main.cpp
│ ├── logistic.cu
│ ├── logistic.h
├── data/
│ └── data.txt
├── .gitignore
├── README.md

## How to Run

- Put your `data.txt` in the `data/` folder
- Compile with Visual Studio or CMake
- Run `main.exe` (ensure working dir is set to project root)
